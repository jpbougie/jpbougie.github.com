Projects

Dam (@github)

A Ruby framework to generate activity streams, such as a facebook news feed. Used in a new project, Crop Circle.

Sift (@github)

Created in the context of my Master's final project at Politecnico di Milano, this is a web tool built in Ruby with Sinatra and CouchDB that offers a gmail-like interface to filter and rate different entries. This was used to fetch questions from the Yahoo! Answers database and explore the features extracted using Natural Language libraries and processes.

Bee (@github)

The backend counterpart of Sift, this is a Scala worker-queue client that makes use of couchdb for the storage of tasks and Kestrel as a queue system to post them.
